#
## Predictive Analytics

### Model Development

A Linear Regression algorithm was applied to estimate transaction revenue.

### Predictor Variables

- Quantity
- UnitPrice
- ItemsInCart

### Response Variable

- TotalPrice

The dataset was divided into training and testing subsets before model construction and evaluation.

---

## Performance Evaluation

### Coefficient of Determination (R²)

**R² = 0.89**

### Interpretation

The model successfully accounts for approximately 88% of the variability observed in transaction revenue. This level of performance indicates that the selected variables provide strong explanatory power and can effectively estimate order values within the dataset.


## Key Insights

- Revenue is strongly influenced by product quantity and pricing.
- Customer purchasing activity changes over time.
- Different product groups contribute unique levels of business value.
- Transaction outcomes can reveal opportunities for operational improvement.
- Predictive models can support revenue estimation and decision-making.
